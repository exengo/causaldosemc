# causaldosemc 0.1.0

- First public GitHub release of the package.
- Includes staged and joint matrix-completion estimators for continuous-
  treatment panel data.
- Includes blocked cross-validation, weighting-aware DR workflows,
  post-fit dose-response estimation, diagnostics, bootstrap inference, and
  sensitivity-analysis helpers.
- Ships with source vignettes for getting started, empirical workflows,
  weighting, dose-response dynamics, and a manuscript-style case study.
